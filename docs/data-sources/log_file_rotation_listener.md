# pingdirectory_log_file_rotation_listener (Data Source)

Describes a Log File Rotation Listener.

## Example Usage

```terraform
data "pingdirectory_log_file_rotation_listener" "myLogFileRotationListener" {
  name = "MyLogFileRotationListener"
}
```
