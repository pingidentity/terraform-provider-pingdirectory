# pingdirectory_custom_logged_stats (Data Source)

Describes a Custom Logged Stats.

## Example Usage

```terraform
data "pingdirectory_custom_logged_stats" "myCustomLoggedStats" {
  name        = "MyCustomLoggedStats"
  plugin_name = "MyPlugin"
}
```
