# pingdirectory_constructed_attribute (Data Source)

Describes a Constructed Attribute.

## Example Usage

terraform {
  required_version = ">=1.1"
  required_providers {

      source  = "pingidentity/pingdirectory"
    }

  product_version        = "9.3.0.0"
